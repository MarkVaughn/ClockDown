ClockDown
=========

Turn upcoming times into funky countdown clocks

You can fiddle here
http://jsfiddle.net/IfTrueElseFalse/QfdEd/34/

parameters

    $('time').clockdown({
        radius: 30
        ringWidth: 0 // if this value is greater than 0 the pie will be a ring, rather than a slice
        borderWidth: 0
    });
